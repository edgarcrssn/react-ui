# A react component library

## Getting started

### Installation
```zsh
npm i @edgarcrssn/react-ui
```

### Usage
Firstly, import the component(s) you wish :
```js
import { Button } from '@edgarcrssn/react-ui';
```

You can now use it in your JSX :
```jsx
<Button color="primary">Click-me!</Button>
```
